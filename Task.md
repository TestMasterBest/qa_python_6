# Финальный проект 6 спринта

В проекте Sprint_6 разработаны автотесты для учебного сервиса "Яндекс.Самокат" с использованием UI-тестирования, Page Object Model и Allure-отчёта.

## Установка проекта
1. Установите браузер Mozilla Firefox для выполнения задания.
2. Клонируйте репозиторий и откройте проект в вашей IDE.
3. Подключите Selenium и Allure к проекту.

## Тестовые сценарии
### Выпадающий список в разделе "Вопросы о важном"
- Необходимо проверить, что при нажатии на стрелочку открывается соответствующий текст. Напишите отдельный тест на каждый вопрос.

### Заказ самоката
1. Проверьте весь флоу позитивного сценария с двумя наборами данных.
2. Проверьте точки входа в сценарий: кнопка "Заказать" вверху страницы и внизу.
3. Позитивный сценарий включает в себя:
    - Нажатие кнопку "Заказать" и проверка появления двух кнопок заказа.
    - Заполнение формы заказа.
    - Проверку появления всплывающего окна с сообщением об успешном создании заказа.
    - Проверку перехода на главную страницу "Самоката" при нажатии на логотип "Самоката".
    - Проверку открытия главной страницы Дзен через редирект при нажатии на логотип Яндекса.

## Написание тестов
1. Опишите необходимые локаторы с помощью Page Object.
2. Создайте отдельный пакет для Page Object и для каждой страницы создайте отдельный класс.
3. Напишите тесты на Selenium, разделив их по тематике или функциональности. Добавьте тесты на одну функциональность в один класс.
4. Все тесты должны лежать в директории test.
5. Используйте параметризацию для различных данных.

## Allure-отчёт
1. Сгенерируйте Allure-отчёт.
2. Запушьте его в репозиторий.
